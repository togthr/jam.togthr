# Contributing

Thanks for helping build Jam Togthr.

## Getting Started

- Read the docs: [`docs/README.md`](docs/README.md)
- Check agent status: [`docs/agent/STATUS.md`](docs/agent/STATUS.md)
- Review workflow expectations: [`docs/agent/WORKFLOW.md`](docs/agent/WORKFLOW.md)

## Workflow

1. Create an issue or task.
2. Work on a branch.
3. Submit a pull request with clear context.

See the issue and pull request templates in [`.github`](.github).

## Code Style

Follow the repository conventions and any instructions in `AGENTS.md` files.

## Security

Report security issues via [`SECURITY.md`](SECURITY.md).
